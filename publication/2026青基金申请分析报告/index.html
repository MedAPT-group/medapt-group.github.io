2026年度国家自然科学基金青年项目（C类）申请策略分析报告

日期: 2026年1月19日

一、成果基础分析

1.1 整体成果概况

根据您提供的publication信息，您的科研成果非常丰富，具有以下特点:

  • 总论文数: 50篇
  • 近3年产出（2023-2025）: 32篇（占比64%），显示出强劲的科研产出能力
  • 顶会顶刊分布:
    • CCF A类会议: CVPR(5), ICCV(3), ECCV(2), NeurIPS(3), ICLR(1), AAAI(2) = 16篇
    • 顶级期刊: TPAMI(2), TIP(5), TMM(1) = 8篇
    • 医学影像专业期刊: Medical Physics(3), Physics and Imaging in Radiation Oncology(4) = 7篇

1.2 研究方向分布

您的研究呈现出明显的交叉融合特征，主要聚焦于:

核心方向1: 医学影像与放射治疗（最强势，37篇）

  • MR引导质子治疗（MR-guided proton therapy）
  • 形变配准（Deformable Image Registration, DIR）
  • MR到CT图像合成（MR-to-CT synthesis）
  • 剂量计算与治疗计划
  • 代表性工作:
    • [MedPhys 2025] Diffusion Schrödinger bridge models for MR-to-CT synthesis
    • [MedPhys 2025] Continuous sPatial-Temporal Deformable Image Registration (CPT-DIR)
    • [PhiRO 2025] Gaussian Representation for Deformable Image Registration
    • [PhiRO 2025] Direct MR-guided dose accumulation

核心方向2: 计算机视觉基础任务（13篇）

  • 图像分割（语义分割、全景分割、参考表达分割）
  • 人体姿态估计（2D/3D pose estimation）
  • 点云分析（3D point cloud understanding）
  • 视觉-语言跨模态（open vocabulary learning）
  • 代表性工作:
    • [CVPR 2024] Skeleton-in-Context: In-Context Learning for skeleton
    • [NeurIPS 2023] Point-In-Context: In-Context Learning for 3D Point Clouds
    • [TPAMI 2024] Towards Open Vocabulary Learning (综述)
    • [CVPR 2024] Neural Clustering based Visual Representation Learning

新兴方向3: In-Context Learning范式（3篇）

  • 这是您在2023-2024年的创新突破点
  • 将NLP的in-context learning引入计算机视觉
  • 应用于骨架序列、点云等多种模态
  • NeurIPS 2023 + CVPR 2024连续顶会，显示该方向的创新性

新兴方向4: 联邦学习（1篇）

  • [NeurIPS 2024] FedLPA: One-shot Federated Learning
  • 展示了对隐私保护机器学习的关注

二、2026年国自然新政策深度解读

2.1 七大改革要点

根据2026年1月19日发布的最新指南，青年基金面临以下重大变革:

改革1: 申请书"瘦身提质"

  • 正文从原来的细分框架简化为三部分:
    1. 立项依据 - 为什么要研究？价值何在？
    2. 研究内容 - 不再预设提纲，自主撰写
    3. 研究基础 - 前期工作积累
  • 篇幅限制: 原则上不超过30页，鼓励简洁表达
  • 影响: 更注重逻辑清晰和重点突出，避免堆砌

改革2: 分阶段评审机制（重要！）

  • 第一阶段: 专家先评估项目创新性，不看研究基础
  • 第二阶段: 再考量申请人前期研究基础
  • 核心目的: 打破"拼履历"困局，给创新性强但积累相对薄弱的项目更多机会
  • 对您的影响:
    • ✅ 有利: 您有丰富的研究积累（50篇论文），第二阶段优势明显
    • ⚠️ 需注意: 第一阶段的创新性论述变得极其关键，必须在立项依据的前500字就抓住评委

改革3: 交叉科学部增设青年C类

  • 交叉科学部新增接收青年科学基金项目（C类）
  • 为交叉领域人才开辟专门资助通道
  • 对您的影响: 您的工作高度交叉（AI+医学影像），可考虑交叉学部申请

改革4: 分类评审持续

  • 继续区分"自由探索类"和"目标导向类"基础研究
  • 需根据研究内容准确选择研究属性

改革5: 经费包干制

  • 青年A/B/C类均实行包干制
  • 申请时无需编制预算，简化申请流程

2.2 新政策下的机遇与挑战

机遇

  1. 创新性优先评审: 如果选题新颖，不会因为"积累不够厚"被一票否决
  2. 交叉学科通道: 医学影像AI是典型的交叉领域，有专门资助渠道
  3. 申请书简化: 减少形式负担，更注重实质内容

挑战

  1. 立项依据前500字成为"生死线": 必须在开篇就说清楚创新点
  2. 研究问题的原创性要求更高: 与现有研究的本质区别必须清晰
  3. 科学突破的潜力需明确阐述: 不能只是技术改进

三、申请方向建议

3.1 推荐方向一: 医学影像AI（最推荐）

研究主题建议

“基于连续表征的智能放疗图像配准与剂量预测方法研究”

为什么推荐这个方向？

  1. 成果基础最强: 37篇论文中最核心的方向，有MedPhys、PhiRO等多篇高质量论文
  2. 创新性突出:
    • CPT-DIR方法将连续表征引入形变配准，突破传统体素方法
    • Gaussian Representation、Diffusion Schrödinger Bridge等新方法
    • 这些工作发表在2025年，正值最新
  3. 临床价值明确: 质子治疗、放疗计划等都有清晰的临床需求
  4. 延续性好: 可以在现有工作基础上深化，不需要从零开始

申请代码选择

主要推荐: H1816（肿瘤放射治疗学）

  • 理由:
    • 您的工作直接服务于放疗临床问题
    • H18（肿瘤学）下的H1816专门对应放射治疗
    • Medical Physics等医学物理期刊发表的工作最匹配此代码

备选1: H2707（医学图像处理）

  • 如果更强调图像处理方法论
  • 适合强调配准、合成等算法创新

备选2: F0604（机器学习）+ 交叉学部

  • 如果强调AI算法的基础研究贡献
  • 可考虑与医学代码交叉申请

创新点提炼（针对分阶段评审）

立项依据前500字应突出:

  1. 现有问题: 传统体素配准在大变形、滑动边界、时空连续性建模上的局限
  2. 本质区别: 提出连续时空表征范式，从离散体素到连续函数空间
  3. 科学突破:
    • 理论: 建立连续时空配准的数学框架
    • 方法: 融合神经隐式表示、扩散模型等前沿技术
    • 应用: 提升质子治疗中器官运动建模的精度和效率

3.2 推荐方向二: In-Context Learning for Medical Imaging

研究主题建议

“医学影像分析的上下文学习方法及其少样本泛化研究”

为什么推荐这个方向？

  1. 创新性强:
    • 您在NeurIPS 2023和CVPR 2024的工作开创了点云/骨架的ICL范式
    • 将此延伸到医学影像是自然且创新的跨界
  2. 前沿性: In-Context Learning是2023-2024年的研究热点
  3. 临床需求: 医学影像标注困难，少样本学习有实际价值
  4. 交叉性: 完美结合CV前沿方法和医学应用

申请代码选择

主要推荐: 交叉科学部

  • 理由:
    • 2026年交叉学部新增青年C类
    • 您的工作高度交叉（CV基础方法论 + 医学应用）
    • 在交叉学部竞争可能相对较小

备选: F0604（机器学习）

  • 强调ICL作为学习范式的基础研究

创新点提炼

  1. 现有问题: 医学影像AI泛化能力差，新任务需重新训练大量标注数据
  2. 本质区别: 从"预训练-微调"范式转向"上下文学习"范式
  3. 科学突破:
    • 建立医学影像ICL的理论框架和基准
    • 解决医学多模态（CT/MRI/PET）的context设计问题
    • 实现零样本或少样本的任务适应

3.3 推荐方向三: 联邦学习+医疗隐私

研究主题建议

“面向多中心医学影像的隐私保护联邦学习方法研究”

为什么推荐这个方向？

  1. 符合国家战略: 数据安全、医疗AI是国家重点方向
  2. 有基础: NeurIPS 2024的FedLPA工作
  3. 临床痛点: 医学数据无法跨机构共享，联邦学习是刚需
  4. 资助热点: 隐私计算、可信AI是近年资助热点

申请代码选择

主要推荐: F0603（网络与信息安全）+ F0604（机器学习）交叉

  • 强调隐私保护和分布式学习

备选: H18 + F06交叉（医学信息学）

创新点提炼

  1. 现有问题: 医学数据孤岛导致AI模型泛化能力受限
  2. 本质区别: 提出后验聚合而非参数聚合的联邦框架
  3. 科学突破: 在保护隐私前提下实现模型性能逼近集中式训练

四、申请代码详细分析

4.1 医学科学部（H18）

  • H1816 肿瘤放射治疗学: 您放疗相关工作的最佳归口
  • H2707 医学图像处理: 适合强调图像配准、合成等算法
  • H2816 生物医学工程: 如果强调工程系统设计

优势:

  • 您在MedPhys、PhiRO等医学期刊有高质量发表
  • 临床价值明确，容易说服医学背景评委

劣势:

  • 医学部评委可能对深度学习方法论不够熟悉
  • 可能更看重临床验证而非算法创新

4.2 信息科学部（F06）

  • F0604 机器学习: 适合强调ICL、联邦学习等算法贡献
  • F0607 计算机视觉: 您CV顶会论文多，这是传统强项

优势:

  • 评委对AI方法论理解深刻
  • CVPR、ICCV、NeurIPS论文在F06认可度高

劣势:

  • 竞争激烈，计算机视觉申请扎堆
  • 医学应用可能被视为"应用研究"而非基础研究

4.3 交叉科学部（2026新增青年C）

适合场景:

  • 方法论创新 + 医学应用深度融合
  • 难以归类到单一学科

优势:

  • 2026年新开放，竞争可能较小
  • 专门为交叉人才设计，匹配您的背景

劣势:

  • 评审标准不确定（新设立）
  • 需要平衡好方法创新和应用价值

五、针对新政策的撰写策略

5.1 立项依据（前500字是关键！）

结构建议:

  1. 第1段（150字）: 问题背景 + 重大挑战

    • 例: “质子治疗中器官运动导致剂量偏差是临床难题，传统体素配准在大变形时失效…”

  2. 第2段（150字）: 现有方法的本质局限

    • 例: “现有方法基于离散体素假设，无法建模时空连续性，导致…”

  3. 第3段（200字）: 本项目的核心创新

    • 例: “本项目提出连续时空表征范式，首次将神经隐式表示引入4D配准，突破…”

避免:

  • ❌ 大篇幅综述现有工作
  • ❌ 过多技术细节
  • ✅ 直击痛点，简洁有力

5.2 研究内容（自主撰写框架）

不要机械套用"研究内容一、二、三"，而应按照逻辑链条组织:

范例框架:

  1. 连续时空配准的理论建模
  2. 基于扩散模型的概率配准方法
  3. 多模态图像合成与配准联合优化
  4. 临床数据验证与系统集成

每部分内容:

  • 科学问题是什么？
  • 拟采用什么方法？
  • 创新点在哪里？
  • 可行性如何？

5.3 研究基础（展示积累但不堆砌）

新政策下的变化:

  • 研究基础在第二阶段才评估
  • 但不意味着不重要，而是要精准匹配

撰写建议:

  1. 选择最相关的5-8篇代表作:

    • 必须与申请方向直接相关
    • 按逻辑线索排列，不是按时间

  2. 突出研究脉络:

    • 例: “我们从2023年开始探索连续表征方法，首次提出Gaussian DIR（PhiRO 2025），进而发展了时空连续建模CPT-DIR（MedPhys 2025），形成了从理论到应用的完整链条…”

  3. 展示能力:

    • 理论建模能力
    • 算法实现能力
    • 临床合作能力
    • 团队支撑能力

避免:

  • ❌ 堆砌所有50篇论文
  • ❌ 列举不相关的工作
  • ✅ 讲清楚"为什么我能做这个研究"

六、综合建议与行动计划

6.1 最优选择（综合评估）

🏆 强烈推荐: 方向一（医学影像AI）+ H1816代码

理由:

  1. ✅ 成果基础最扎实: 37篇论文，MedPhys 3篇 + PhiRO 4篇
  2. ✅ 创新性充分: CPT-DIR、Gaussian DIR等2025年最新工作展示了持续创新能力
  3. ✅ 临床价值清晰: 质子治疗是高端医疗，国家重视
  4. ✅ 符合新政策: 连续表征是真正的方法论创新，不是简单技术改进
  5. ✅ 竞争相对合理: 医学影像AI在H1816申请人数相比F06少

6.2 备选方案

  • 适合条件: 如果您想强调基础方法论贡献，愿意承担新学部的不确定性
  • 优势: 创新性极强，NeurIPS+CVPR背书
  • 风险: 交叉学部第一年，评审标准未知

备选B: 方向三（联邦学习+医疗隐私）+ F0603/F0604

  • 优势: 符合国家战略（数据安全）
  • 风险: FedLPA只有1篇，基础相对薄弱

6.4 注意事项

    • 可行性要有前期工作支撑（第二阶段评审）

  4. 准备充分的前期数据:

    • 初步实验结果
    • 临床合作证明（如有）
    • 关键技术的原理验证

七、2026年国自然政策要点总结

关键信息

  • 申请截止: 2026年3月20日16:00
  • 申请书调整: 三部分结构（立项依据+研究内容+研究基础），≤30页
  • 分阶段评审: 创新性优先，研究基础后评
  • 交叉学部: 新增青年C类接收
  • 经费: 青年C类包干制

  3. 简洁性: 30页讲清楚，不堆砌
  4. 匹配度: 前期工作与申请内容高度一致
